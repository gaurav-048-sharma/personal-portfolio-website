MongoDB

MongoDB is a NoSQL, document-oriented database. Unlike traditional relational databases that store data in tables and rows, MongoDB stores data in JSON-like documents called BSON (Binary JSON). Each document can have a flexible schema, which makes MongoDB highly adaptable and scalable.

React

React.js is a JavaScript library (not a framework) for building user interfaces, especially single-page applications (SPAs). It was developed by Facebook and is component-based, declarative, and highly efficient for rendering UI.

Node JS

A runtime environment that allows you to run JavaScript on the server-side. It’s built on Chrome’s V8 engine and used to build scalable backend services like APIs.
